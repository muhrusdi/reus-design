# Button

Easy-to use button components.

## Type

<p>
  To define the type of button the directive is used <code>type</code> with the value of some type of button like it can be: <code>primary</code>.
</p>

## API

<p>
  To get a customized button component, just set 
  <code>
    type
  </code>, 
  <code>
    size
  </code>, 
  <code>
    disabled
  </code>.
</p>

| Property                  | Default | Description |
| ------------------------- | ------- | -----------
| type           | default | 
| size           | default |
| disabled           | default |
